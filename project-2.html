No-Proxy-Attendance

A hassle-free solution for marking attendance within college network. The QR Code Attendance System is an efficient,fast and user-friendly tool for tracking attendance using QR codes. It utilizes HTML, CSS, and Flask to create a web-based interface for marking attendance. This system is designed to work seamlessly when devices are connected to the Same College Local Network.

Live Link

Project Overview

The QR Code Attendance System employs HTML, CSS, and Flask, facilitating efficient attendance tracking on a web interface. It automatically fetches the IPv4 address, generates QR codes, and allows real-time attendance marking within the same local network. The Faculty Panel enables easy management of attendance records.

The QR Code Attendance System utilizes HTML, CSS, and Flask to create a user-friendly web interface for attendance tracking. It automatically fetches the IPv4 address to generate QR codes for local network connections. The Faculty Panel allows teachers to manage attendance effectively, removing duplicates. Users access the system at http://localhost:8000, scan QR codes for real-time attendance updates, and benefit from a straightforward interface for efficient attendance management.
