Project Overview

The project is a feedback system for citizens to assess police station services, accessible through mobile web browsers. It employs HTML, CSS, JavaScript, PHP, Java, and databases. Police officers use a web/app interface to analyze service quality based on survey questions. The system aids in understanding public needs, improving conditions, and ultimately enhancing services, contributing to a reduction in crime rates and better community policing.

A login page incorporating Google's reCAPTCHA for verification. Users enter credentials, and the reCAPTCHA ensures human interaction. Upon successful verification, the script redirects to a home page. The project aims to enhance citizen-police interaction with a mobile web interface, enabling feedback. Police officers access a comprehensive web/app interface using HTML, CSS, and JavaScript, backed by PHP and Java for database functionality. Survey questions aid in service quality analysis, addressing community needs, ultimately contributing to crime reduction and improved public services.
